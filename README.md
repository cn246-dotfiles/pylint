## pylint Dotfiles

`pylint` configuration

----

### Installing
- Create a hidden dotfiles directory:
```bash
mkdir -p "${HOME}/.dotfiles"
```

- Change directories:
```bash
cd "${HOME}/.dotfiles"
```

- Clone this repo
```bash
git clone git@github.com:cn246-dotfiles/less.git
```

- Stow the dotfiles:
```bash
stow --verbose --no-folding pylint
```
